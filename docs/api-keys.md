# API Key Checklist

- `GOOGLE_GENAI_API_KEY`: Create in Google AI Studio (https://aistudio.google.com/app/apikey)
- `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY`, `SUPABASE_SERVICE_ROLE_KEY`: Project settings in Supabase (https://app.supabase.com)
- `TAVILY_API_KEY`: Tavily dashboard (https://app.tavily.com)
- `BRAVE_SEARCH_API_KEY`: Brave Search API dashboard (https://api-dashboard.search.brave.com)
- `REDDIT_CLIENT_ID`, `REDDIT_CLIENT_SECRET`, `REDDIT_USER_AGENT`: Reddit app registration (https://www.reddit.com/prefs/apps)
