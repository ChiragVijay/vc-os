A Next.js application with three integrated tools for venture capital portfolio management and founder self-benchmarking.


### 1. **Diligence Agent** (`/diligence`)

AI-powered due diligence research for evaluating potential investments.

- Natural language research queries
- Real-time streaming report generation with citations
- Structured analysis: market, technology, team, traction, risks
- Source index with clickable references
- Export to Markdown and PDF formats

### 2. **Portfolio Dashboard** (`/dashboard`)

Analytics and benchmarking for existing investments.

- Portfolio-wide KPIs (ARR, growth, runway)
- Percentile rankings across metrics
- Batch cohort comparisons
- At-risk company alerts
- Company deep-dives with 13-month history

### 3. **Founder Portal** (`/founder`)

Self-service metrics and benchmarking for portfolio founders.

- Pre-loaded demo mode
- Metric input with auto-calculations
- Anonymized percentile benchmarks
- 6-month projections (conservative/current/optimistic)
- Fundraising readiness checklist
- Health score breakdown

## Getting Started

```bash
bun install
bun dev
```
